Multi-Site Enterprise Network

ARCHITECTURE OVERVIEW

Architected and simulated a highly secure multi-site enterprise network topology within a GNS3 virtualized environment. Designed to connect international corporate hubs while maintaining strict perimeter defense.

INFRASTRUCTURE IMPLEMENTATION

  • Global Connectivity: Established stable VPN/IPSec site-to-site tunnels connecting nodes across Jordan, Saudi Arabia, and Turkey with a designated Disaster Recovery (DR) site.
  • Layered Security Controls: Implemented ASA firewalls with DMZ, port security, DHCP security, and SSH hardening.
  • Access Management: Configured core services and access control lists (ACLs) to enforce least privilege between VLANs and remote sites.

Luminex Diagnostic Dashboard

THE BIOMEDICAL CHALLENGE

Modern medical laboratories rely on complex, highly sensitive diagnostic hardware (such as Luminex FM3D & 200). Standard IT infrastructure tools often fail to meet the strict security and operational requirements of these closed networks.

TECHNICAL SOLUTION

  • Development of an interactive troubleshooting dashboard.
  • Real-time operational monitoring for lab technicians.
  • Integration of Mokafih AI-driven assistants for anomaly detection.

SECURITY COMPLIANCE

Architecting the deployment environment to securely handle highly sensitive technical and diagnostic data without exposing the internal lab network to external threat vectors.
